# 🤖 ИНСТРУКЦИЯ ПО ЗАПУСКУ БОТА «Идентичность 360°»

## ШАГ 1 — Заполни настройки в файле bot.py

Открой файл `bot.py` и найди в самом начале две строки:

```
BOT_TOKEN = "ВАШ_ТОКЕН_ЗДЕСЬ"
ADMIN_CHAT_ID = "ВАШ_ID_ЗДЕСЬ"
```

Замени:
- `ВАШ_ТОКЕН_ЗДЕСЬ` → токен от @BotFather (например: `7412638590:AAFx3kB...`)
- `ВАШ_ID_ЗДЕСЬ` → твой числовой ID от @userinfobot (например: `123456789`)

---

## ШАГ 2 — Установи Python

Проверь, есть ли Python на компьютере:
- Открой Командную строку (Windows) или Терминал (Mac)
- Напиши: `python --version`
- Если видишь версию 3.8+ — отлично, переходи дальше
- Если нет — скачай Python с сайта https://python.org (кнопка Download)

---

## ШАГ 3 — Установи библиотеку

В терминале перейди в папку с ботом и выполни:

```
pip install -r requirements.txt
```

---

## ШАГ 4 — Запусти бота

```
python bot.py
```

Если видишь `✅ Бот запущен!` — всё работает!
Открой своего бота в Telegram и напиши `/start`

---

## ШАГ 5 — Бот в интернете (чтобы работал всегда)

Пока бот запущен только на твоём компьютере.
Чтобы он работал 24/7 — нужен хостинг.

### Бесплатный вариант — Railway.app:

1. Зайди на https://railway.app
2. Зарегистрируйся через GitHub (нужен аккаунт)
3. Нажми «New Project» → «Deploy from GitHub repo»
4. Загрузи папку с ботом
5. В разделе Variables добавь:
   - `BOT_TOKEN` = твой токен
   - `ADMIN_CHAT_ID` = твой ID
6. Railway запустит бота автоматически

---

## Что делает бот

✅ Проводит клиентку через 7 блоков анкеты
✅ Принимает фотографии
✅ Сохраняет все ответы в файл `answers.txt`
✅ Отправляет тебе уведомление когда клиентка завершила
✅ Работает с любым количеством клиенток одновременно

## Команды бота

- `/start` — начать анкету
- `/cancel` — прервать и начать заново
